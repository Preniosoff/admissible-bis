Espace élève

Savoir quoi faire maintenant

L’espace élève réduit le travail à une action claire : comprendre une notion, s’entraîner, reprendre une erreur ou suivre l’avancement.

Action prioritaire

Si vous arrivez sans savoir par où commencer, générez trois priorités courtes pour aujourd’hui. Le but est de sortir avec une trace concrète, pas de tout ouvrir.

Créer un compte guidé Priorités du jour

Si je suis perdu

Choisissez d’abord le niveau, puis un besoin. Une recherche libre arrive seulement après, quand la notion ou le chapitre est clair.

Choisir mon niveau → Rechercher avec filtres →

Parcours recommandé

  1. 01Choisir le niveau. Collège, lycée ou classes préparatoires.
  2. 02Choisir le besoin. Comprendre, s’entraîner, préparer une évaluation ou reprendre une erreur.
  3. 03Agir sur une seule ressource. Fiche de cours, méthode ou exercice ciblé.
  4. 04Garder une trace. Notion comprise, exercice corrigé ou erreur reformulée.
